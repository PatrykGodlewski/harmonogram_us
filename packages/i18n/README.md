# @repo/i18n

Shared Paraglide + inlang project for **`apps/web`** and **`apps/admin`**.

- **`project.inlang/settings.json`** ‚Äî locales, plugins, message file pattern.
- **`messages/*.json`** ‚Äî translation source files.
- **`src/paraglide/`** ‚Äî generated by Paraglide (gitignored; produced on `pnpm install` / `pnpm run compile`).

Apps import:

- `@repo/i18n/paraglide/runtime`
- `@repo/i18n/paraglide/messages`

Regenerate after editing `messages/`:

```bash
pnpm --filter @repo/i18n run compile
```

From the repo root this also runs as `pnpm run messages:compile`.
